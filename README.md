# Virtual Company

Autonomous company simulation with CEO + 3 managers + 7 employees and 25 skills across sales, ops, tech, and exec.

## Live Demo

- Web app: https://crgatade-beep.github.io/virtual-company
- Landing page: https://crgatade-beep.github.io/virtual-company

## Structure

- `virtual_company.html`: main web app
- `company/company_policies.md`: company rules and policy
- `company/environment_setup.md`: environment setup and roles
- `tasks/tasks.json`: task definitions and schedules
- `logs/`: execution logs and scorecards

## Run locally

1. Open `virtual_company/html` in a browser.
2. Optional: use `company_sim.py` for headless/auto runs.

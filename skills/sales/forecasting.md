## Definition
Predicting future sales revenue by analyzing historical data, current pipeline, and market trends to set realistic targets and allocate resources.

## Methods
1. **Historical forecasting**: Extend past trends
2. **Pipeline forecasting**: Weighted by deal probability and stage
3. **Intuitive forecasting**: Sales rep estimates (least accurate)
4. **AI-augmented forecasting**: LLM-assisted pattern recognition

## Weighted Pipeline Formula
```
Forecast = Σ(Deal Value × Stage Probability)
Stage probabilities:
- Lead: 5%
- Qualified: 15%
- Proposal: 40%
- Negotiation: 70%
- Closed: 100%
```

## Best Practices
- Update weekly, not monthly
- Compare forecast vs actual quarterly
- Include seasonal patterns
- Factor in macro economic changes
- Present 3 scenarios: optimistic, realistic, pessimistic

## KPIs
- Forecast accuracy: ±10%
- Pipeline coverage ratio: 3x (3x forecast value in pipeline)
- Weighted forecast: Updated weekly
- Variance explanation: For gaps >15%

## Improvement Protocol
When variance > 15%: run root cause analysis, check stage probabilities accuracy, audit data quality.

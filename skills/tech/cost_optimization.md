## Definition
Continuous monitoring and optimization of technology spend (cloud infra, SaaS tools, licenses) to maximize output per dollar spent.

## Cloud Cost Levers
1. **Right-sizing**: Match instance size to workload
2. **Auto-scaling**: Scale down during off-hours
3. **Spot/preemptible**: Use for fault-tolerant, batch workloads
4. **Reserved instances**: 1-3 year commitments for steady-state
5. **Storage tiers**: Move cold data to cheaper tiers
6. **Data transfer**: Reduce cross-region/egress costs

## SaaS Optimization
- Audit all tools quarterly: Are we using all seats?
- Negotiate annually for 20-30% discount
- Consolidate overlapping tools
- Remove unused licenses within 30 days of detection

## Cost Tracking
- Tag all resources by department/project
- Set budgets per team with alerts at 80/95%
- Weekly cost review in Ops standup
- Monthly report to CEO

## KPIs
- Cloud waste: < 15%
- SaaS utilization: 90%+
- Cost per feature shipped: Down trend
- Annual savings reinvested in growth: 100%

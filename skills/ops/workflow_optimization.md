## Definition
Systematic analysis and improvement of business processes to reduce waste, speed up throughput, and improve quality without increasing cost.

## Methodology
1. **MAP**: Document current state (flowchart, swimlane diagram)
2. **MEASURE**: Collect baseline data (cycle time, error rate, cost per unit)
3. **ANALYZE**: Identify bottlenecks, rework loops, waiting times
4. **IMPROVE**: Design future state, implement changes
5. **CONTROL**: Monitor new process, sustain gains

## Waste Types (Lean)
1. **Transport**: Unnecessary movement of info/items
2. **Inventory**: Excess stock/WIP
3. **Motion**: Excess effort by people
4. **Waiting**: Idle time between steps
5. **Overprocessing**: Steps that add no customer value
6. **Overproduction**: More than needed
7. **Defects**: Rework, scrap, corrections

## Optimization Techniques
- Parallel processing (non-dependent steps run simultaneously)
- Batch reduction (smaller batches = faster feedback)
- Automation triggers (if X > N times, automate)
- Error-proofing (poka-yoke — make mistakes impossible)

## KPIs
- Cycle time reduction: 20%+
- Error rate reduction: 30%+
- Cost per unit: Down
- Throughput: Up 25%+
